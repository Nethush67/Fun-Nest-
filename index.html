Common Issues

App Not Working / Crashes

If the app isn’t starting or behaving unexpectedly, try reinstalling it. Please note that reinstalling may reset settings.

Settings Reset / Nothing Saves

Some settings might not persist if the app is closed during an update or crash. Make sure the app fully loads before closing.

Connection Issues

If certain features aren’t working, such as syncing or games like the Cookie Clicker mini-game, ensure your Wi-Fi is connected and stable. Reconnect if necessary.

Feature-Specific Problems

If a particular feature, like Cookie Clicker, is broken, reinstalling the app often resolves the issue.
